GCAT with LMM library

General computational analysis tool with linear mixed model library

This is a library for GCAT https://github.com/danny-wilson/gcat-project

It provides tools for linear mixed model analysis

This source code allows GCAT to be run with the LMM library using Docker and Singularity. A pre-built image is available from https://hub.docker.com/r/dannywilson/gcat-lmm/
